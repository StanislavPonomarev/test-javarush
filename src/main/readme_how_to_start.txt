Для создания тестовой базы данных выполнить в схеме (root@root) mysql скрипт create and fill database user.sql ,
и этот скрипт создаст тестовую базу TEST, в ней таблицу USER, заполнит в таблице первые 7 записей.
Далее,в файле config.properties по необходимости отредактировать путь к созданной вами базе TEST, логин и пароль 
(переменные *.url , *.username, *.password).
Затем, создать в IDE новый проект maven, скопировать в его каталог main все папки данной программы, откомпилировать java src,
и можно делать maven install на tomcat, запустить tomcat. Всё, программа должна работать
